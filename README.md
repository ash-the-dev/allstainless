# All Stainless – Business Website

A modern website built for a stainless steel fabrication and installation company.  
The goal was simple: create a clean, fast, and trustworthy online presence that actually converts visitors into customers.

## Live Sites
- https://allstainlessinstallations.com  
- https://allstainless.vercel.app

## What I built
- Fully responsive layout for mobile, tablet, and desktop
- Clear service structure so customers can quickly understand offerings
- Contact-focused design to drive inquiries
- Performance-optimized Next.js setup

## Tech Stack
- Next.js
- React
- CSS / custom styling
- Vercel for deployment

## Why this project matters
This wasn’t a practice project. It was built to represent a real business and needed to look credible to actual customers.  
The focus was on clarity, speed, and making sure nothing got in the way of someone reaching out.
